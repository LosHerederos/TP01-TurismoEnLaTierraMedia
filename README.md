## Cómo clonar este repositorio

Lo primero es obtener la URL del repositorio

![Copiar URL del repo en GitHub](_imgs/guia_clonar_repo_url_de_repo_en_github.png)

### Clonar con GitBash

Abrir GitBash e ir a la ubicación de nuestro workspace de Eclipse.

![](_imgs/guia_clonar_repo_con_gitbash-01.png)

Ejecutar el comando `git clone <url_repositorio>`

![](_imgs/guia_clonar_repo_con_gitbash-02.png)

Repositorio clonado

![](_imgs/guia_clonar_repo_con_gitbash-03.png)

Abrir e importar el proyecto en Eclipse

![](_imgs/guia_clonar_repo_con_gitbash-04.png)

Seleccionar la opción 'General/Existing Projects into Workspace' y presionar Next

![](_imgs/guia_clonar_repo_con_gitbash-05.png)

Presionar el boton Browse 

![](_imgs/guia_clonar_repo_con_gitbash-06.png)

Seleccionar la carpeta del proyecto

![](_imgs/guia_clonar_repo_con_gitbash-07.png)

Dar clic en Finish

![](_imgs/guia_clonar_repo_con_gitbash-08.png)


### Clonar con Eclipse

Abrir Eclipse e ir al menu File/Import

![](_imgs/guia_clonar_repo_con_eclipse-01.png)

Seleccionar la opción 'Git/Projects from Git' y presionar Next

![](_imgs/guia_clonar_repo_con_eclipse-02.png)

Seleccionar la opción 'Clone URI' y presionar Next

![](_imgs/guia_clonar_repo_con_eclipse-03.png)

En el campo URI pegar la URL del repo

![](_imgs/guia_clonar_repo_con_eclipse-04.png)

Completar los datos de Authentication y presionar Next

![](_imgs/guia_clonar_repo_con_eclipse-05.png)

Dejar todo como esta y darle a Next

![](_imgs/guia_clonar_repo_con_eclipse-06.png)

Presionar el boton browse y seleccionar la ubicación de nuestro workspace de Eclipse

![](_imgs/guia_clonar_repo_con_eclipse-07.png)


![](_imgs/guia_clonar_repo_con_eclipse-08.png)

Dejar todo como esta y darle a Next

![](_imgs/guia_clonar_repo_con_eclipse-09.png)

Dar clic en Finish

![](_imgs/guia_clonar_repo_con_eclipse-10.png)
